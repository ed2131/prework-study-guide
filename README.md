# Bootcamp Prework Study Guide Webpage

## Description

Provides a concise aggregate of the new terms, definitions and concepts presented in the bootcamp prework. It covers topics on HTML, CSS, git and Javascript and condenses the information in one place.

- This webpage is designed to be a quick reference to be refered to frequently
- It provides convenient access to basic information in a clear condensed format
- New terms encountered in the bootcamp are conveniently defined and explained 
- The webpage in interactive and is itself constructed using concepts covered in the prework
